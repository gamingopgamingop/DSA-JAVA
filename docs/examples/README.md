# Examples and Input/Output

This directory contains practical examples, input/output samples, and dry runs for all algorithms and data structures in the DSA Java project.

## 📁 Directory Structure

### Algorithm Examples
- **[sorting/](sorting/)** - Sorting algorithm examples
- **[searching/](searching/)** - Search algorithm examples
- **[data-structures/](data-structures/)** - Data structure operation examples

### Input/Output Files
- **[INPUT_OUTPUT_EXAMPLES.md](INPUT_OUTPUT_EXAMPLES.md)** - Comprehensive I/O examples
- **[dry-runs/](dry-runs/)** - Step-by-step algorithm execution
- **[sample-data/](sample-data/)** - Sample input files

## 🎯 Usage Examples

### Running Examples
```bash
# Compile and run with sample input
javac AlgorithmName.java
java AlgorithmName < sample-data/input.txt

# View expected output
cat sample-data/output.txt
```

### Example Categories
1. **Basic Usage**: Simple input/output examples
2. **Edge Cases**: Boundary condition examples
3. **Performance**: Large dataset examples
4. **Error Cases**: Invalid input examples
