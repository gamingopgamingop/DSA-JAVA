- **Educational Value**: Progressive complexity

### Implementation Focus
```java
// Bubble Sort
for (int i = 0; i < n-1; i++) {
    for (int j = 0; j < n-i-1; j++) {
        if (arr[j] > arr[j+1]) {
            int temp = arr[j];
            arr[j] = arr[j+1];
            arr[j+1] = temp;
        }
    }
}

// Selection Sort
for (int i = 0; i < n-1; i++) {
    int minIdx = i;
    for (int j = i+1; j < n; j++) {
        if (arr[j] < arr[minIdx]) {
            minIdx = j;
        }
    }
    // Swap minIdx with i
}
```

### Key Differences
- **Bubble Sort**: Simple, stable, educational
- **Insertion Sort**: Adaptive for nearly sorted data
- **Selection Sort**: Non-stable, simple implementation
- **Merge Sort**: Stable, consistent performance

### Implementation Focus
```java
// Merge Sort
public static void mergeSort(int[] arr, int left, int right) {
    if (left < right) {
        int mid = (left + right) / 2;
        mergeSort(arr, left, mid);
        mergeSort(arr, mid + 1, right);
        merge(arr, left, mid, right);
    }
}

// Quick Sort
public static void quickSort(int[] arr, int low, int high) {
    if (low < high) {
        int pi = partition(arr, low, high);
        quickSort(arr, low, pi - 1);
        quickSort(arr, pi + 1, high);
    }
}
```

### Key Differences
- **Merge Sort**: Stable, consistent O(n log n)
- **Quick Sort**: Faster average case, O(n²) worst case
- **Memory**: Merge Sort O(n), Quick Sort O(log n)
- **Applications**: Different use cases

```java
// Hybrid Quick Sort
public static void hybridQuickSort(int[] arr, int left, int right) {
    // Use insertion sort for small subarrays
    if (right - left <= 16) {
        insertionSort(arr, left, right);
    } else {
        quickSort(arr, left, right);
    }
}

// Parallel Merge Sort
public static void parallelMergeSort(int[] arr, int left, int right) {
    // Parallel merge implementation
    ForkJoinPool pool = new ForkJoinPool();
    int mid = (left + right) / 2;
    ForkJoinPool.invokeAll(() -> {
        merge(arr, left, mid, mid);
        return merge(arr, mid + 1, right);
    }, pool);
}
```
